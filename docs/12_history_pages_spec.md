# 12_history_pages_spec.md - 記録履歴閲覧ページ 要件定義

> ステータス: 仕様策定中（UIデザイン・詳細レイアウトは別途決定）

## 背景・課題

現状、活動記録（`/records/activity`）と個別観察記録は最新10〜20件しか参照できない。
日付・クラス・児童名などの条件で過去の記録を検索・閲覧できるページが存在しない。

## 追加するページ

| ページ | URL |
|--------|-----|
| 活動記録履歴 | `/records/activity/history` |
| 個別観察記録履歴 | `/records/personal/history` |

---

## 共通仕様

### タブナビゲーション

- 両ページの上部に「活動記録」「個別記録」のタブを配置する
- タブ押下で対応するページに遷移する
- **共通の検索条件（日付範囲・クラス）はタブ切り替え時に引き継ぐ**
  - 例：活動記録で「2025年1月・ひまわり組」で絞っていた場合、個別記録タブに切り替えても同条件が適用された状態になる
  - 片方にしかない条件（個別記録の「児童名」など）は引き継ぎ対象外

### デフォルト表示

- ページを開いた時点で、**登録日降順の最新20件**を取得・表示する
- 検索条件を入力すると、その条件で絞り込まれた結果を表示する（最新20件から再取得）

### 追加読み込み（もっと見る）

- 一覧の末尾に「もっと見る」ボタンを配置する
- ボタン押下でさらに20件を追加取得し、現在の一覧の下に追記する
- 取得可能な残件数がゼロになったらボタンを非表示にする
- 1ページあたりの件数: **20件**（`limit=20`, `offset`をインクリメント）

## 活動記録履歴ページ（`/records/activity/history`）

### 検索・絞り込み条件

| 条件 | 入力形式 | 備考 |
|------|---------|------|
| 日付範囲（開始） | 日付ピッカー | 未入力時は制限なし |
| 日付範囲（終了） | 日付ピッカー | 未入力時は制限なし |
| クラス | セレクトボックス | 「すべて」を含む |
| 担当者（記録したスタッフ） | セレクトボックス or テキスト検索 | 「すべて」を含む |
| キーワード | テキスト入力 | 活動内容・引き継ぎ等のテキスト全文検索 |

### 一覧表示項目（1件あたり）

- 記録日
- クラス名
- タイトル（イベント名 or 活動内容の冒頭）
- 担当者名
- 個別記録の紐づき件数

---

## 個別観察記録履歴ページ（`/records/personal/history`）

### 検索・絞り込み条件

| 条件 | 入力形式 | 備考 |
|------|---------|------|
| 日付範囲（開始） | 日付ピッカー | 未入力時は制限なし |
| 日付範囲（終了） | 日付ピッカー | 未入力時は制限なし |
| クラス | セレクトボックス | 「すべて」を含む |
| 担当者（記録したスタッフ） | セレクトボックス or テキスト検索 | 「すべて」を含む |
| 児童名 | テキスト入力（かな対応） | 部分一致 |
| カテゴリ（タグ） | セレクトボックス or マルチセレクト | 「すべて」を含む |
| キーワード | テキスト入力 | 観察内容のテキスト全文検索 |

### 一覧表示項目（1件あたり）

- 記録日
- 児童名
- クラス名
- カテゴリ（タグ）
- 観察内容の冒頭テキスト
- 担当者名
